3. Modelamos

Aplicamos el modelo de kNN con 5 vecinos y distancia Euclidiana para valores puramente numericos, con weighted vote a las 2 versiones del dataset, normalizado y sin normalizar para ver que performance obtenemos. Normalizado, obtenemos un 96.23% de precisión

Tras comparar la performance de ambos modelos, se puede concluir que el que se entrenó y testeó con datos normalizados resultó tener una exactitud sumamente superior al otro y muestra lo importante que es la normalización en kNN donde las distancias juegan un rol protagónico.
