Predicción de segundo ataque Cardiaco - Logistic Regression con sklearn vs Rapidminer

do. 01 octubre 2023

En este notebook compararemos los resultados de un modelo de regresion logistica utilizando sklearn y Rapidminer sobre el mismo dataset,

Analizaremos el problema de predecir un segundo ataque cardíaco dadas ciertas condiciones medicas iniciales luego del primero.

Para hacerlo contamos con 2 datasets, training y scoring, el desarrollo de este notebook será el siguiente:

  1. En primera instancia trabajaremos con sklearn y dividiremos el dataset de training en 2 para ver como se desempeña el algoritmo de logistic regression para el problema, esto es porque en el dataset de training contamos la información clasificada y podemos evaluar performance.

  2. Luego entrenaremos el algoritmo con todo el dataset de training y le ingestaremos el dataset de scoring, tanto en RapidMiner como en sklearn para ver sus predicciones.

  3. Compararemos los resultados de las predicciones de RapidMiner y sklearn para ver que tan de acuerdo están.

1. Analizamos el desempeño de logistic regression para nuestro problema con sklearn

import matplotlib

from sklearn.model_selection import train_test_split

#funcion auxiliar para cargar dataframe desde un CSV en Google Drive
def loadCSVFromGoogleDrive(url):
  file_id=url.split('/')[-2]
  dwn_url='https://drive.google.com/uc?id=' + file_id
  df = pd.read_csv(dwn_url) #cargamos dataframe
  return df
